Comandos principales (desde la raíz):
  npm run backend     # Inicia el servidor API
  npm run vue:serve   # Dev server Vue (puerto 5173 por defecto)
  npm run build       # Genera frontend/dist/

Notas:
- public/ en la raíz es la fuente única de img/, sounds/ y favicon.ico.
- Express sirve /img y /sounds desde public/ (ver src/app.js).
- El frontend legacy HTML/JS fue retirado; no existe carpeta frontend/js.
